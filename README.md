# Home TODO

Household task manager for families. Mobile-first PWA.

**Stack:** React 19 · TypeScript · Vite · Tailwind CSS 4 · localStorage

**Live:** [lamp2022.github.io/home-todo](https://lamp2022.github.io/home-todo/)

## Dev

```sh
npm install
npm run dev      # localhost:5173
npm test         # vitest
npm run build    # production build
```

## Features

- Quick task creation with natural language deadlines (Finnish)
- Category and time-based views with grouping
- Recurring tasks (weekly/monthly/yearly)
- Multi-person assignment
- Swipe-to-close bottom sheet
- Mobile-optimized (44px touch targets, safe areas, no iOS zoom)
